Instructions how to install and use the Easy Report Tool
========================================================

The steps:

1. Extarct files from the archive.

2. Choose the projct folder on C:\ --> AFCON --> Pulse  

3. Choose which data to save.

4. Click on "Create Report" and wait for a message box to finish the process.

5. Go to C:\AFCON and look for 2 newly created files SupportReport.zip and Support folder.
